# 정말 간단한 [ Node.js - if ] 조건식

| 조건식  |                   설명                   |
| :-----: | :--------------------------------------: |
|  a < b  |           a가 b보다 작으면 참            |
|  a > b  |            a가 b보다 크면 참             |
| a != b  |            a와 b가 다르면 참             |
| a <= b  |        a가 b보다 작거나 같으면 참        |
| a >= b  |        a가 b보다 크거나 같으면 참        |
| a == b  | a와 b가 같으면 참 ( 자료형은 달라도 됨 ) |
| a === b | a와 b가 같으면 참 ( 자료형도 같아야 함 ) |
